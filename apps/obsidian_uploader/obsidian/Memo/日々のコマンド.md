# パッケージのアップデート

## Mac

```shell
# homebrew
brew update
brew upgrade
brew cleanup

# zsh
zimfw update

# Rust
rustup update
cargo install-update -a

# mise
mise up

# uv
uv self update

# npm tools
npm update -g
```

## Linux (VPS)

```shell
# dnf
sudo dnf update
sudo dnf autoremove

# Rust
rustup update
cargo install-update -a

# mise
mise up
```

## Windows

```shell
# winget
winget upgrade --all

# rust
rustup update
cargo install-update -a
```

# プロジェクト内で使用しているパッケージのアップデート

## Bun

```shell
bun outdated
bun update
bun update --latest
```

## Cargo

```shell
cargo outdated
cargo update
```
